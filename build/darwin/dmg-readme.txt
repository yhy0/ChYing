macOS Installation / macOS 安装说明
=====================================

Drag chying.app to Applications folder to install.
将 chying.app 拖到 Applications 文件夹即可安装。

If macOS shows "chying is damaged and can't be opened":
如果 macOS 提示"chying 已损坏，无法打开"：

Open Terminal and run / 打开终端执行：

    sudo xattr -cr /Applications/chying.app

Then open chying.app again.
然后重新打开 chying.app 即可。
